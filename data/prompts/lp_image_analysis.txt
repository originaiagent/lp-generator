このLP（ランディングページ）画像を詳細に分析してください。

【最初に判定】
まず、この画像が以下のどれに該当するか判定してください：
A. LPのページ（ファーストビュー、機能説明、口コミなど）
B. 商品パッケージ、説明書、同梱物の写真
C. 商品単体の写真（白背景など）
D. その他（判別不能）

B、C、Dの場合は、以下のJSON形式で出力して分析を終了してください：
{
  "page_type": "商品画像" または "パッケージ" または "その他",
  "is_lp_page": false,
  "skip_reason": "LPページではないためスキップ",
  "brief_description": "画像の簡単な説明（1文）"
}

Aの場合のみ、以下の詳細分析を行ってください：

【分析項目】
1. ページ種別（ファーストビュー/機能説明/権威付け/比較表/口コミ/CTA/その他）
2. 含まれるテキスト要素をすべて抽出（種類、内容、文字数、狙い）
3. ビジュアル要素（写真、イラスト、装飾）
4. デザインの特徴・配色・レイアウト
5. このページの狙い・役割

【テキスト要素の種類】
- メインキャッチコピー: 一番目立つ見出し
- サブヘッド: 補足的な見出し
- リードコピー: 導入文
- ブレット: 箇書きの項目（item_countで項目数を記録）
- キーワード群: グリッド等で並ぶ短いキーワード（item_countで項目数を記録）
- 権威付け: 専門家監修、受賞歴など
- 注意書き: 免責事項など

【出力形式】JSON
{
  "page_type": "ページ種別",
  "is_lp_page": true,
  "main_message": "このページが伝えたいこと（1文）",
  "texts": [
    {"type": "種類", "content": "実際のテキスト内容", "char_count": 文字数, "aim": "狙い・効果"},
    {"type": "ブレット", "items": ["項目1", "項目2"], "item_count": 2, "char_per_item": 平均文字数, "aim": "狙い・効果"},
    {"type": "キーワード群", "items": ["KW1", "KW2", "KW3"], "item_count": 3, "char_per_item": 平均文字数, "aim": "狙い・効果"}
  ],
  "visuals": [
    {"type": "写真/イラスト/装飾", "description": "説明", "purpose": "目的"}
  ],
  "design_features": "デザインの特徴（配色、レイアウト、印象）",
  "page_role": "LP全体におけるこのページの役割"
}

【厳守事項】
- 画像に実際に写っていない文字は絶対に出力しないこと。推測や補完は禁止。
- テキスト要素のcontentには画像内の実際のテキストを正確に書き起こすこと
- char_countは実際の文字数を数えること
